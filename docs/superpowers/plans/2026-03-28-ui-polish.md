# Grove UI Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Elevate the visual quality of the Grove UI — redesign the hero/home page with depth and suggestion chips, remove score badges from result rows, and add a type-colored left accent + CSS-driven action fade to results.

**Architecture:** Pure frontend changes in three files — `App.tsx` (hero section layout, meta row), `ResultList.tsx` (score removal, CSS action visibility, type accent), `index.css` (new utility classes for all of the above). No backend, no new components, no new dependencies.

**Tech Stack:** React 19, TypeScript, Tailwind CSS, JetBrains Mono + Inter fonts, CSS custom properties, `color-mix()` (supported in Tauri v2's Chromium WebView).
